# Spatial Joins, Point-in-Polygon Counts & Kernel Density

Graduate coursework — Emory MPH, BIOS 532 (Spatial Analysis). Aggregating point events to areas and visualizing spatial intensity.

## What it covers
- Spatial joins with `st_join`
- Counting cholera deaths within Soho neighborhood polygons (point-in-polygon)
- Kernel density estimation heatmaps with `SpatialKDE` and `terra`

## Tools
R · Quarto · `sf` · `tmap` · `rio` · `SpatialKDE` · `terra`

## Data
John Snow cholera period-2 deaths, water pumps, neighborhoods, and Soho census (`data/`).

## How to run
Open `project.Rproj` in RStudio and render `spatial.qmd`. Rendered output: `spatial.html`.
